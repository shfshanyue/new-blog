Cursor 最近越来越慢了，经常写着写着就卡住一下。我打开 IDE 的次数也变少了，但 Cursor 还是用得很频繁。

因为我挪到命令行使用 cursor-agent 来写代码，类似 claude code 与 amp。图片粘贴，skills 支持地也非常好，与 Cursor IDE 体感差别挺夸张，内容飞速生成。

如果你也被卡顿折磨，可以试试使用 cursor-agent 来代替 cursor 的 IDE，体验会好很多。
