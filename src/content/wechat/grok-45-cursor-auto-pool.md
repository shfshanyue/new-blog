最近有两个模型发布，一个是 Grok 4.5，一个是 gpt-5.6。对我个人来说，更喜欢、也更让我振奋的是 Grok 4.5。

虽然 Grok 4.5 的编程能力并不是很强，但对我这个 Cursor 用户来说是一个比较大的利好。因为它是一个比 Composer 更强的模型，而且它与 Composer 一起使用 Auto 的池子，刚好解决了 Cursor 没有前沿模型的一个弊端。

虽然 Cursor 也可以使用 Opus 4.8 等模型，但是在 API 池子中，额度非常非常有限，因为那些是其他公司的模型。而 Grok 4.5 是 Cursor 已经被 SpaceX 收购，这是和 SpaceXAI 共同研发出来的，所以成本较低，额度限制会少很多。

有一个要注意的点：在 Cursor 里使用 Grok 4.5 时，一定要注意是不是 Fast 模式，Fast 价格更贵。
