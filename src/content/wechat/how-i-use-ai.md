简单说下我目前怎么用 AI。

豆包在手机端和浏览器插件上用得特别多，深度且高频率。

工作中主要用 Cursor 的 Compose，以及 Claude 的 Opus 和 Sonnet。自动化这边是 Cursor Automation，比较轻量且便宜。结合定时任务的话，一般跟编程相关的都基于 Cursor Automation，无关的基于 Manus 的定时任务。

之前试过龙虾，Token 花费比较贵，目前已经放弃了。最近 Hermes 比较火，我有可能去试一试；若尝试的话，可能会购买一些国产模型的付费计划。但决定条件在于是否有充足的需求，并且使用 Cursor Automation 以及 Manus 定时任务以及 Use Computer 也不能完成。

自动化尽量做到全自动化，而没办法全自动化的，尽可能是最大程度地半自动化。但半自动化和全自动化的差距很大，就像斗圣巅峰与斗帝。
