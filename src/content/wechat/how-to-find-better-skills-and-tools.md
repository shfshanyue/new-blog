最近我经常用一个叫 Website Audit 的 skill。它做的事很简单：跑一条命令，把生成的报告分析给你看。说白了，这个 skill 就是那个命令行工具的一层壳。

这样的 skill 不少。比如

1. agent browser skill，本质就是 agent-browser 命令行工具的封装
2. find skills skill，现在下载量很大，它也是 npx skills-find 的壳。

问题就来了：怎么找到更好的 skill？

有时候可以换个思路：怎么找到更好用的命令和工具。把思路从「找 skill」换成「找好 cli」，反而更容易下手。
