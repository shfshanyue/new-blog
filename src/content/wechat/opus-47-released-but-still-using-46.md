opus 4.7 发布了。

发布的时候我大概看了一眼，能力提升了，但价格也涨了。然后我就关掉页面，继续用 4.6 了。

不是不想用新的。主要是我现在跑了一套完整的工作流，代码生成、review、文档整理，每个环节都跑通了。这种情况下，换模型这件事对我的影响其实没那么大——工作流本身做了很多事，模型只是其中一个节点，换不换区别没想象的那么大。

再一个是 4.7 贵了。我不是说贵一点点，是明显贵了一档。如果换过去，每个月的 Token 消耗算下来肉疼，而且现在 4.6 做的事情还没让我觉得不够用。

还有一个原因是 4.7 不太说人话。我这说的不是技术能力，是它有时候回答问题喜欢绕弯子，解释一堆背景，才把你要的东西给你。4.6 更直接，问什么给什么，读起来没那么累。

所以我现在的状态就是：知道 4.7 发布了，也知道它更强，但就是没有切换的动力。等哪天 4.6 明显不够用了，或者 4.7 的价格调整了，再说吧。
