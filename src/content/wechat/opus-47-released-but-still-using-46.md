opus 4.7 发布了，最近几天在 Cursor 上试了试，因为有半价优惠，价格没那么肉疼。

试完之后我还是切回 4.6 了。

不是不好用。主要是我现在跑了一套完整的工作流，头脑风暴、生成 Spec、生成 Plan、执行 Plan、Review，每个环节都跑通了。这种情况下，换模型这件事对我的影响其实没那么大——工作流本身做了很多事，模型只是其中一个节点，换不换区别没想象的那么大。

再一个是 4.7 价格标着和 4.6 一样，但它换了新的 Tokenizer，同样一段内容消耗的 Token 可以多出 10% 到 35%，算下来还是贵了一截。

还有一个原因是 4.7 不太说人话。
