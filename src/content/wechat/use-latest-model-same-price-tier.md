不要使用过期或者退役的模型。

不少人会以为，模型出得晚、能力更强，价格肯定更贵。其实同一档位的模型，价格差不多。

以 Claude Sonnet 为例：Sonnet 4.0、4.5、4.6 定价是一样的。如果你还在用 4.0，成本跟用 4.6 没区别，但能力差一截，多少有点亏。

以 GPT 为例：有的新模型，比如 GPT 5.0，反而比 GPT-4o 更便宜。所以别默认「新的就一定贵」，看一眼价目表再选。

如果你不需要很强能力的模型，价格还想便宜些，那就选 gpt 的 nano 模型，比如 gpt-5-nano。

openai 的价格表：https://developers.openai.com/api/docs/pricing

cursor 列的价格表：https://cursor.com/docs/models 所有模型都会列上
